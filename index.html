Selamat Datang di VALORANT

VALORANT adalah game FPS (First Person Shooter) taktis buatan Riot Games yang menggabungkan kemampuan karakter (Agent) dengan strategi tim.

Penjelasan Game VALORANT

VALORANT merupakan game 5v5 di mana pemain terbagi menjadi dua tim, yaitu Attacker dan Defender. Setiap pemain memilih Agent dengan skill unik.

Tujuan Game VALORANT

  • Mencapai 13 ronde kemenangan
  • Attacker menanam Spike
  • Defender menggagalkan atau menjinakkan Spike
  • Menghabisi seluruh tim lawan
